# SoftwareAgent-AggregatorManager-AndroidClient
The project consists of a Java client (Software Agent), a Java server (Aggregator Manager) and an Android app (Android Client).
User uses the app to send Nmap commands to the server. Server sends the commands to the Software Agent in order to execute them.
We use RESTful web services for communication. Contains Javadoc. Written in Java and Android using Eclipse and Android Studio.
